Tópicos do HTML

<title>

A tag <title> é usada para definir o título da página que é exibido na barra de título do navegador. É uma tag obrigatória em qualquer documento HTML.

<h1> - <h6>

As tags de cabeçalho <h1> - <h6> são usadas para definir os diferentes níveis de títulos em uma página. O <h1> é o mais importante, enquanto o <h6> é o menos importante.

<p>

A tag <p> é usada para definir um parágrafo de texto. É uma das tags mais comumente usadas para estruturar o conteúdo de uma página.
