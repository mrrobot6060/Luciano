Tópicos de HTML e Python

Tópicos de HTML

<title>

O elemento <title> define o título do documento HTML.

<script>

O elemento <script> é usado para inserir ou referenciar um script em um documento HTML.

<h1> - <h6>

Os elementos de cabeçalho <h1> a <h6> são usados para definir cabeçalhos de seção.

Tópicos de Python

input()

A função input() é usada para receber uma entrada de usuário.

print()

A função print() é usada para exibir uma saída na tela.

while

A declaração while é usada para criar um loop que executa repetidamente um bloco de código enquanto uma condição especificada for verdadeira.

int()

A função int() é usada para converter um valor em um número inteiro.

if, elif, else

As declarações if, elif e else são usadas para tomar decisões condicionais em um programa Python.

<p>

O elemento <p> é usado para definir um parágrafo.

<a>

O elemento <a> é usado para criar um link para outra página.

<img>

O elemento <img> é usado para exibir uma imagem em um documento HTML.

<ul> e <li>

Os elementos <ul> e <li> são usados para criar listas não ordenadas.

for

A declaração for é usada para criar um loop que itera sobre uma sequência de elementos.

list()

A função list() é usada para criar uma lista em Python.

len()

A função len() é usada para obter o comprimento (número de elementos) de uma sequência ou coleção.

range()

A função range() é usada para criar uma sequência de números.

def

A declaração def é usada para definir uma função em Python.

<div>

O elemento <div> é usado para criar uma divisão ou seção em um documento HTML.

<form>

O elemento <form> é usado para criar um formulário em um documento HTML para entrada de dados.

<input>

O elemento <input> é usado para criar campos de entrada em um formulário HTML.

<table>, <tr> e <td>

Os elementos <table>, <tr> e <td> são usados para criar tabelas em um documento HTML.

str()

A função str() é usada para converter um valor em uma string em Python.

list comprehension

A list comprehension é uma forma concisa de criar listas em Python usando uma única linha de código.

dict()

A função dict() é usada para criar um dicionário em Python.

try, except

As declarações try e except são usadas para tratar exceções (erros) em um programa Python.

<head>

O elemento <head> é usado para incluir informações do cabeçalho do documento HTML, como metadados, scripts e folhas de estilo.

<body>

O elemento <body> é usado para definir o corpo principal do documento HTML.

<div>

O elemento <div> é usado para agrupar elementos HTML e aplicar estilos ou manipular com JavaScript.

<span>

O elemento <span> é usado para aplicar estilos ou manipular partes específicas do texto dentro de um elemento.

<ul> e <ol>

Os elementos <ul> e <ol> são usados para criar listas não ordenadas e ordenadas, respectivamente.

<input> (tipos: text, checkbox, radio)

O elemento <input> é usado para criar campos de entrada em um formulário HTML, com diferentes tipos para capturar texto, caixas de seleção e botões de rádio.

<img>

O elemento <img> é usado para exibir uma imagem em um documento HTML.

<a>

O elemento <a> é usado para criar links para outras páginas ou recursos.

<table>

O elemento <table> é usado para criar tabelas em um documento HTML.

<form>

O elemento <form> é usado para criar formulários interativos em um documento HTML.

Listas

Listas são estruturas de dados usadas para armazenar vários itens em uma única variável.

Tuplas

Tuplas são sequências imutáveis de elementos em Python.

Dicionários

Dicionários são estruturas de dados usadas para armazenar pares chave-valor em Python.

Funções

Funções são blocos de código reutilizáveis projetados para realizar uma tarefa específica.

Módulos

Módulos são arquivos que contêm definições de funções, classes e variáveis que podem ser usados em outros programas Python.

Manipulação de arquivos

Python fornece recursos para criar, ler, gravar e manipular arquivos de dados.

Tratamento de exceções

Tratamento de exceções é o processo de lidar com erros e exceções durante a execução de um programa Python.

Expressões regulares

As expressões regulares são sequências de caracteres usadas para correspondência de padrões em texto.

<hgroup>

O elemento <hgroup> é usado para agrupar vários elementos de cabeçalho (<h1> a <h6>) em um único cabeçalho.

<meta>

O elemento <meta> é usado para especificar metadados sobre um documento HTML, como codificação de caracteres, descrição e palavras-chave.

<iframe>

O elemento <iframe> é usado para incorporar conteúdo de outra fonte em uma página HTML.

<nav>

O elemento <nav> é usado para definir uma seção de navegação em um documento HTML.

<button>

O elemento <button> é usado para criar um botão clicável em um documento HTML.

<label>

O elemento <label> é usado para associar um texto descritivo a um elemento de formulário em um documento HTML.

<textarea>

O elemento <textarea> é usado para criar uma área de texto multi-linha em um formulário HTML.

<select> e <option>

Os elementos <select> e <option> são usados para criar listas suspensas (dropdown) em um formulário HTML.

Herança

Herança é um conceito de programação orientada a objetos que permite que uma classe herde atributos e métodos de outra classe.

Polimorfismo

Polimorfismo é um conceito de programação orientada a objetos que permite que objetos de diferentes classes sejam tratados de forma semelhante.

Arquivos CSV

Arquivos CSV (Comma-Separated Values) são usados para armazenar dados em formato tabular, onde cada valor é separado por uma vírgula.

Bibliotecas externas

Python possui uma ampla variedade de bibliotecas externas que estendem suas funcionalidades, permitindo tarefas como processamento de imagem, análise de dados, geração de gráficos, entre outros.

Manipulação de data e hora

Python possui bibliotecas para manipular datas, horários e realizar operações relacionadas, como formatação, cálculos e conversões.

Recursão

A recursão é um conceito em programação onde uma função chama a si mesma para resolver um problema.

Gerenciamento de pacotes com pip

O pip é uma ferramenta de linha de comando usada para instalar, atualizar e gerenciar pacotes externos em Python.

<h1>

O elemento <h1> é usado para definir o cabeçalho de nível 1 em um documento HTML.

<h1>Título do Documento</h1>

<p>

O elemento <p> é usado para definir um parágrafo em um documento HTML.

<p>Este é um parágrafo de exemplo.</p>

Print

O comando print() é usado para exibir uma mensagem ou valor na saída.

print("Olá, mundo!")

If

A declaração if é usada para executar um bloco de código se uma condição for verdadeira.

idade = 18 if idade >= 18: print("Você é maior de idade.")

Lista

Uma lista é uma coleção ordenada de elementos em Python.

frutas = ["maçã", "banana", "laranja"] print(frutas)

<a>

O elemento <a> é usado para criar um link para outra página ou recurso.

<a href="https://www.example.com">Visite o exemplo.com</a>

<img>

O elemento <img> é usado para exibir uma imagem em um documento HTML.

<img src="imagem.jpg" alt="Imagem de exemplo">

Funções

Uma função é um bloco de código reutilizável que realiza uma tarefa específica.

def saudacao(nome): print("Olá, " + nome + "! Bem-vindo(a)!") saudacao("João")

For loop

O loop for é usado para iterar sobre uma sequência de elementos em Python.

frutas = ["maçã", "banana", "laranja"] for fruta in frutas: print(fruta)

<form>

O elemento <form> é usado para criar um formulário interativo em um documento HTML.

<form action="/processar" method="post"> <input type="text" name="nome" placeholder="Digite seu nome"> <input type="submit" value="Enviar"> </form>

<table>

O elemento <table> é usado para criar uma tabela em um documento HTML.

<table> <tr> <th>Nome</th> <th>Idade</th> </tr> <tr> <td>João</td> <td>25</td> </tr> </table>

Importação de Módulos

A importação de módulos permite o uso de funcionalidades adicionais em Python.

import math print(math.sqrt(16))

While loop

O loop while é usado para repetir um bloco de código enquanto uma condição for verdadeira.

i = 0 while i < 5: print(i) i += 1

<div>

O elemento <div> é usado para agrupar elementos em um documento HTML e aplicar estilos ou manipulações.

<div> <h2>Título do Bloco</h2> <p>Conteúdo do bloco.</p> </div>

Tuplas

Uma tupla é uma sequência imutável de elementos em Python.

ponto = (10, 20) print(ponto[0])
