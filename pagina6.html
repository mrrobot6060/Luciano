HTML Avançado

Elementos de Mídia

O HTML oferece suporte a vários elementos de mídia para exibir conteúdo multimídia. Aqui estão alguns exemplos:

<video>

Para exibir um vídeo, use o elemento <video>. Por exemplo:

            <video src="video.mp4" controls>
                Seu navegador não suporta o elemento <video>.
            </video>

<audio>

Para reproduzir um áudio, use o elemento <audio>. Por exemplo:

            <audio src="audio.mp3" controls>
                Seu navegador não suporta o elemento <audio>.
            </audio>

Elementos Semânticos

Os elementos semânticos fornecem um significado estrutural ao conteúdo da página. Eles ajudam os mecanismos de pesquisa a entender melhor o seu conteúdo. Aqui estão alguns exemplos:

<header>

O elemento <header> representa o cabeçalho da página ou de uma seção. Por exemplo:

            <header>
                <h1>Título da Página</h1>
                <p>Descrição do site</p>
            </header>

<article>

O elemento <article> representa um conteúdo independente que faz sentido por si só. Por exemplo:

            <article>
                <h2>Título do Artigo</h2>
                                
Conteúdo do artigo...

<section>

O elemento <section> representa uma seção lógica ou temática de conteúdo. Por exemplo:

            <section>
                <h2>Seção 1</h2>
                <p>Conteúdo da seção 1...</p>
            </section>

Elementos de Formulário

Os elementos de formulário permitem a coleta de dados do usuário. Aqui estão alguns exemplos:

<input>

O elemento <input> é usado para criar campos de entrada. Por exemplo:

            <input type="text" name="nome" placeholder="Seu nome">

<textarea>

O elemento <textarea> é usado para criar campos de texto multilinha. Por exemplo:

            <textarea name="mensagem" rows="4" cols="50">
                Digite sua mensagem aqui...
            </textarea>

Conclusão

Esses são apenas alguns exemplos de elementos HTML avançados. Existem muitos outros recursos que você pode explorar para criar páginas da web mais sofisticadas e interativas.
