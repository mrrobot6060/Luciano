Resumo da Estrutura HTML

O HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo de uma página da web. Aqui está um exemplo básico de estrutura HTML:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Título da Página</title>
        </head>
        <body>
            <h1>Título Principal</h1>
            <p>Parágrafo de exemplo.</p>
        </body>
        </html>

No exemplo acima, temos a estrutura básica do documento HTML. Aqui estão alguns conceitos importantes:

<!DOCTYPE html>

A declaração <!DOCTYPE html> define o tipo de documento como HTML5.

<html>

A tag <html> é o elemento raiz de um documento HTML e contém todos os outros elementos.

<head>

A tag <head> contém informações sobre o documento, como o título da página, metadados e links para arquivos externos.

<title>

A tag <title> define o título da página, que é exibido na barra de título do navegador.

<body>

A tag <body> contém todo o conteúdo visível da página, como texto, imagens, links e elementos estruturais.

<h1>, <p> e outros elementos

Os elementos HTML, como <h1> (cabeçalho), <p>Os elementos HTML, como <h1> (cabeçalho), <p> (parágrafo) e muitos outros, são usados para estruturar e formatar o conteúdo da página. Aqui estão alguns exemplos:

<a>

O elemento <a> é usado para criar links. Por exemplo:

            <a href="https://www.exemplo.com">Link Exemplo</a>

<img>

O elemento <img> é usado para exibir imagens na página. Por exemplo:

            <img src="imagem.jpg" alt="Descrição da imagem">

<ul> e <li>

Os elementos <ul> e <li> são usados para criar listas não ordenadas. Por exemplo:

            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ul>

<table>, <tr> e <td>

Os elementos <table>, <tr> e <td> são usados para criar tabelas. Por exemplo:

            <table>
                <tr>
                    <td>Dado 1</td>
                    <td>Dado 2</td>
                </tr>
                <tr>
                    <td>Dado 3</td>
                    <td>Dado 4</td>
                </tr>
            </table>

<form> e <input>

Os elementos <form> e <input> são usados para criar formulários interativos. Por exemplo:

            <form action="processar.php" method="post">
                <input type="text" name="nome" placeholder="Digite seu nome">
                <input type="submit" value="Enviar">
            </form>

<div> e <span>

Os elementos <div> e <span> são usados para agrupar e estilizar outros elementos. Por exemplo:

            <div class="container">
                <h2>Título do Contêiner</h2>
                <p>Texto de exemplo</p>
            </div>

<div> e <span>

Os elementos <div> e <span> são usados para agrupar e estilizar outros elementos. Por exemplo:

            <div class="container">
                <h2>Título do Contêiner</h2>
                <p>Texto de exemplo</p>
            </div>

Esses são apenas alguns exemplos de elementos HTML. Existem muitos outros que você pode explorar para criar páginas da web ricas em conteúdo e interatividade.

Continue praticando e explorando os recursos do HTML para aprimorar suas habilidades de programação web!
