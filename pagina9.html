Tutorial de Python - Exercícios

Exercício 1

Crie um programa que solicite ao usuário um número e exiba a tabuada desse número.

Exercício 2

Escreva um programa que receba uma lista de números do usuário e exiba apenas os números pares.

Exercício 3

Crie uma função que verifique se uma palavra é um palíndromo (lê-se da mesma forma de trás para frente).

Exercício 4

Desenvolva um programa que leia um arquivo de texto e conte quantas vezes uma determinada palavra aparece.

Exercício 5

Crie uma função que calcule a média dos valores de uma lista.
